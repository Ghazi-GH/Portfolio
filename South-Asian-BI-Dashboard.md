# South Asian Macroeconomic Volatility Tracker (Power BI)

## 🎯 The Problem
Cross-country macroeconomic comparisons are often buried in dense, static PDF reports from the World Bank or IMF. Policymakers and asset managers lack an intuitive, interactive interface to track regional trade vulnerabilities, inflation spikes, and currency volatility metrics simultaneously across South Asian economies.

## 🛠️ The Technical Action
* Engineered a dynamic data pipeline consolidating key macroeconomic indicators for South Asian countries.
* Developed custom business intelligence layouts focused on structural minimalism, replacing cluttered default visuals with clean, high-density charts.
* Utilized advanced **DAX (Data Analysis Expressions)** to build time-series volatility indexes and automated year-over-year variance tracking.
* Designed a geographic regional map interface that lets users cross-filter trade performance metrics across different fiscal years instantly.

## 🚀 The Result
Created an executive-level Power BI dashboard that translates complex regional data into a compelling, interactive visual story, significantly reducing the time required to spot cross-border economic trends.
